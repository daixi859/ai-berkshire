## 一、这是一门什么生意？

中际联合是全球领先的高空安全作业设备及服务解决方案提供商，核心业务聚焦**风电行业高空安全设备**。简而言之，这是一家**卖"安全"给风电场**的公司。

### 1.1 核心产品矩阵

| 产品类别 | 用途 | 单价区间（估） | 生命周期 |
|----------|------|---------------|---------|
| 免爬器（爬梯助力器） | 帮助运维人员沿塔筒内部爬梯上下，电动助力 | 数万元/套 | 8-12年 |
| 塔筒升降机 | 搭载人员/轻型货物垂直运输 | 10-30万元/套 | 15-20年 |
| 海上平台吊机 | 海上风机平台物料吊装 | 数十万元/套 | 15-20年 |
| 安全防护设备（防坠落系统等） | 高空作业人员安全防护 | 数千-数万元 | 5-8年 |
| 高空作业服务 | 安装、维保、检测、改造升级 | 按次/按年 | 持续 |

### 1.2 收入结构拆解

#### 按业务类别

| 业务板块 | 收入占比（2025年报） | 毛利率特征 | 增长驱动 |
|----------|-------------------|-----------|---------|
| 专用高空安全作业设备 | **95.67%** | ~47-50%，高毛利 | 新装机+存量改造 |
| 高空安全作业服务 | **3.71%** | 推测高于设备，以人为核心 | 存量风机增长 |
| 其他业务 | 0.62% | — | — |

**核心发现**：目前公司收入极度依赖设备销售（95.67%），服务收入仅占3.71%。这是"产品型"生意而非"平台型"生意。但从长生命周期看，随着全球装机存量扩大，服务收入占比有望逐步提升。

#### 按市场区域

> 注：公司披露产品覆盖75个国家和地区，但公开数据未精确拆分国内外收入比例。据招股书及行业报告估算：

| 区域 | 收入占比（估） | 特征 |
|------|--------------|------|
| 中国国内 | ~60-65% | 核心市场，存量竞争加剧 |
| 海外（欧洲、北美、亚太等） | ~35-40% | 高增长，高毛利率，认证门槛高 |

海外市场是关键的增量来源。欧洲和北美对高空安全标准要求极高，认证壁垒构成天然的"高价保护区"。

#### 按业务驱动：新装机 vs 存量改造

```
                    新装机市场
                  /            \
                 /              \
    全球风电新增装机量 → 设备需求 → 中际联合收入
                 \              /
                  \            /
                    存量改造市场
                    （更稳定、更持续）
```

- **新装机驱动**：全球风电年度新增装机容量（GW级）直接拉动新设备需求。2024-2025年全球风电装机加速增长，这是公司2025年收入大增44.6%的核心原因之一。
- **存量改造驱动**：全球已安装的风机超过40万台（截至2025年），其中大量早期安装的爬梯设备无电动助力或不符合最新安全标准，存量改造空间巨大。这是一个"慢但持续"的需求。

### 1.3 商业模式本质判断

> **段永平说"好生意"：简单、清晰、可理解。**

中际联合的商业模式本质上可以这样理解：

**"给全球每一座风力发电机装上安全升降设备，并持续维护。"**

这门生意的核心逻辑：
- **卖点**：安全合规（不是可选项，是刚需）
- **客户**：集中度高（全球风电运营商约50-100家大型客户）
- **支付能力**：强（风电运营现金流好，安全设备占总投资比例极低）
- **复购率**：高（存量运维、升级改造、新装机配套）

这门生意**不复杂**——非常符合段永平"简单易懂"的标准。

## 二、飞轮效应分析

### 2.1 是否存在飞轮效应？

中际联合的飞轮与传统互联网飞轮不同，它更多是**工业品的规模飞轮**：

```
全球市占率第一
      ↓
生产规模扩大 → 单位成本降低
      ↓
性价比提升/利润空间增大
      ↓
更多研发投入 → 产品力增强
      ↓
更强的品牌信任 → 更多客户
      ↓
    全球市占率第一（闭环）
```

### 2.2 飞轮的关键支点

| 飞轮要素 | 存在度 | 解释 |
|----------|--------|------|
| **网络效应** | ★☆☆☆☆ 不存在 | 安全设备不是社交产品，一个客户的使用不影响其他客户的价值。不存在经典的网络效应。 |
| **规模效应** | ★★★★☆ 强 | 全球市占率第一（细分市场），在采购、生产、物流、服务网络上有显著规模优势。95.67%收入来自设备，规模化生产直接降低单位成本。 |
| **品牌飞轮** | ★★★★☆ 较强 | 安全设备领域，"事故记录"就是品牌。零事故/低事故记录本身在加速积累。每多装一台，品牌信任就多一分。 |
| **数据飞轮** | ★★☆☆☆ 初步 | 10,000+台设备运行数据可反哺产品迭代，但尚未形成核心壁垒。 |

### 2.3 飞轮效应强度总评

**这是一个中等强度的工业品规模飞轮，缺乏网络效应的指数级爆发力，但胜在稳定、可持续、可预测。** 这种飞轮在段永平的体系中可能更受青睐——因为它"可靠"而非"性感"。

## 三、护城河分析（逐一验证）

### 3.1 品牌/声誉护城河 ★★★★☆

| 维度 | 判断 | 分析 |
|------|------|------|
| 安全记录积累 | 强壁垒 | 高空安全设备的**第一要义是不出事**。中际联合深耕20年，产品遍布75国，安全记录本身就是最强的品牌资产。一个新进入者即使技术达标，也需要时间证明自己不会出事。 |
| 国家级认证 | 强壁垒 | 国家专精特新"小巨人"、工信部"制造业单项冠军示范企业"——这些认证是官方背书的护城河。 |
| 国际认证 | 极强壁垒 | 风电高空设备要通过CE（欧盟）、ANSI（美国）、CSA（加拿大）等认证，周期长、成本高。中际联合已全部通过，新进入者需要2-3年才能完成。 |
| 客户信任 | 中等壁垒 | 风电运营商**不会为了省10%的成本而冒险换一家安全设备供应商**——人命关天。这种心理壁垒极高。 |

**结论**：品牌信任是强大的护城河。在安全设备行业，"用过没问题"就是最好的销售。中际联合的20年安全运行记录，是所有竞争对手最难复制的东西。

### 3.2 转换成本 ★★★★☆

| 客户行为 | 转换成本 | 分析 |
|---------|---------|------|
| 新项目选型 | 低→中 | 新风电项目招标时，中际联合面对竞争。但一旦中标，后续转换成本逐步升高。 |
| 已经安装设备的改造 | 高 | 免爬器和升降机通常嵌入塔筒结构设计，更换需要停机、改造结构，成本远高于原厂维保升级。 |
| 日常维保 | 中→高 | 维保团队熟悉自家设备的每一个机械部件。换供应商意味着培训新人、适配新设备，存在安全风险。 |
| 备件供应 | 非常高 | 非标设备，备件不通用。换供应商意味着整个备件体系重构。 |

**具体量化估算**：
- 一套塔筒升降机的采购成本：10-30万元
- 更换供应商的工程改造成本：采购价的50-100%
- 因停机造成的发电损失：每天数万至数十万元
- **总转换成本：采购价的2-3倍以上**

**结论**：转换成本非常高。一旦中际联合的设备"镶嵌"在风机塔筒里，客户基本上就绑定在这个生态系统里了。这是段永平最喜欢的护城河类型之一。

### 3.3 网络效应 ★☆☆☆☆

**不存在经典网络效应**。这是一个"单点产品"而非"平台生意"。客户之间没有互动价值，不存在"越多客户越有价值"的飞轮。

但存在一种**间接网络效应**：
- 设备安装量越大 → 服务网络越密 → 响应速度越快 → 新客户更愿意选择

这种效应在To B工业品中常见，但强度远不及消费互联网网络效应。

### 3.4 规模效应 ★★★★☆

| 维度 | 分析 |
|------|------|
| 采购端 | 全球最大采购量，对钢材、电机、电子元器件等原材料有议价权 |
| 生产端 | 产能利用率高，固定成本被摊薄（2025年毛利率47.3%，同比提升1.8个百分点） |
| 研发端 | 研发费用约7900万元（2025年）/9500万元（2023年），占收入4-8%，摊薄后成本可控 |
| 服务端 | 全球75国服务网络一旦建成，新增客户的边际服务成本极低 |
| 物流端 | 大规模出货可优化海运、陆运成本 |

**具体数据佐证**：
- 2023→2025年，收入增长70%，毛利率从44.9%提升至47.3%——规模效应在毛利率上体现
- 净利率从18.7%提升至28.2%——费用刚性，收入增长带来利润弹性

### 3.5 技术/专利壁垒 ★★★☆☆

| 维度 | 评估 | 分析 |
|------|------|------|
| 专利数量 | 中等 | 高空安全设备领域有相当的专利布局，但核心技术壁垒不如芯片/制药 |
| 认证壁垒 | 高 | 全球各国安全认证是强大的进入壁垒，新进入者需2-3年获取 |
| Know-how | 高 | 20年积累的工艺know-how、客户现场经验、安全数据，难以复制 |
| 技术颠覆风险 | 低 | 高空安全设备是成熟技术，技术路线稳定，颠覆性创新概率低 |

**结论**：技术壁垒中等，但**认证壁垒+know-how壁垒形成有效组合**。单项来看不够强，但组合起来构成了进入障碍。

### 3.6 护城河综合评分

| 护城河维度 | 评分 | 段永平视角评语 |
|-----------|------|--------------|
| 品牌/声誉 | ★★★★ | "安全设备，不出事就是最好的品牌。" |
| 转换成本 | ★★★★ | "装上去就下不来了——客户不想换。" |
| 网络效应 | ★ | "这不是社交网络，别想太多。" |
| 规模效应 | ★★★★ | "全球第一，成本优势是真实的。" |
| 技术/认证壁垒 | ★★★ | "认证是门槛，但不是万年不变的东西。" |
| **综合** | **★★★★** | **一道由安全记录、转换成本、全球认证组成的复合护城河。** |

## 四、用户/客户价值分析

### 4.1 为客户创造了什么独特价值？

| 价值维度 | 具体体现 |
|----------|---------|
| **安全** | 将风机运维人员从"爬100米高塔"的极高风险工作中解放出来。塔筒升降机和免爬器将坠落风险降低90%以上。 |
| **效率** | 升降机将攀爬时间从20-30分钟缩短到2-3分钟，大幅提升运维效率。 |
| **合规** | 全球各国对高空作业安全的监管日益严格（欧盟EN 1808、美国ANSI等）。中际联合的设备本身就是"合规通行证"。 |
| **全生命周期成本** | 虽然采购价高于部分竞争对手，但更高的可靠性+更低的维护成本+20年使用寿命，总持有成本更低。 |
| **一站式采购** | 从免爬器到升降机到防坠落系统到海上吊机，客户可以从单一供应商获得全套解决方案。 |

### 4.2 客户为什么愿意付高价？

> **段永平说："差异化就是客户愿意为你多付钱。"**

中际联合的议价能力来源于：
1. **安全溢价**：安全设备不是成本中心，而是"保险"——风电运营商愿意多付10-20%采购品牌设备，换取"不出事"的确定性
2. **合规溢价**：没有认证的设备根本无法进入欧洲和北美市场 —— 这是法定的"强买强卖"
3. **停机成本倒挂**：一台5MW风机停机一天的发电损失约5-10万元，而一套升降机采购价才10-30万元。两三天的发电损失就够买一套设备了。

### 4.3 客户粘性判断

| 指标 | 判断 |
|------|------|
| 客户集中度 | 中等（前5大客户占比约30-40%，但全球Top 10风电运营商大多是其客户） |
| 客户切换意愿 | 低（安全风险、改造成本、备件体系） |
| 复购模式 | 每新增一台风机就新购一套设备；存量设备每8-15年换代一次 |
| 服务续约 | 年维保合同，续约率应该很高（未公开披露） |

## 五、段永平"好生意"标准评估

### 5.1 是否有差异化？★★★★☆

**是，且差异化深刻。**

| 差异化维度 | 程度 | 解释 |
|-----------|------|------|
| 产品差异化 | 中 | 功能层面差异不大，但"可靠"本身就是差异化 |
| 品牌差异化 | 强 | 安全设备行业，品牌即信任，信任即差异化 |
| 服务差异化 | 中 | 全球服务网络，响应速度 |
| 认证差异化 | 很强 | 全球市场准入是硬差异化——没有认证就是不能卖 |

### 5.2 是否有定价权？★★★★☆

**有较强的定价权，但非绝对。**

- **国内市场**：定价权中等，面对国内竞争对手有一定价格压力，但品牌溢价仍然存在
- **海外市场**：定价权强，认证壁垒+品牌信任构成高溢价能力
- **整体趋势**：随着海外收入占比提升，整体定价权在增强

**关键测试**：如果中际联合涨价10%，客户会换供应商吗？
- 对于已安装设备范畴（服务、备件、改造）：**大概率不会换**——转换成本远高于涨价金额
- 对于新项目：**有一定可能性会考虑竞品**，但安全记录和认证壁垒仍然构成保护

### 5.3 是否有可持续竞争优势？★★★★☆

**护城河可持续，但有需要注意的地方。**

优势来源的可持续性评估：

| 护城河来源 | 可持续性 | 风险点 |
|-----------|---------|-------|
| 品牌/声誉 | 非常可持续 | 只要不出重大安全事故，只会越来越厚 |
| 转换成本 | 非常可持续 | 物理嵌入，本质不可逆 |
| 规模效应 | 可持续 | 需要持续管理产能利用率 |
| 认证壁垒 | 中等可持续 | 竞争对手可以通过时间+投入逐步获取认证 |

**整体护城河增量趋势**：正向增强。每多卖一台设备、每多服务一年，品牌信任和转换成本都在增加。这是典型的"越老越值钱"的护城河。

### 5.4 生意模式是否简单易懂？★★★★★

**极度简单易。——这是段永平最喜欢的那一类。**

可以在一句话内讲清楚：
> "给风力发电机装电梯和安全梯的公司，全球做得最好。"

没有金融杠杆、没有复杂对赌、没有库存减值风险（订单式生产）、没有应收款大雷（客户是现金流充沛的电力运营商）。

### 5.5 长期前景如何？★★★★☆

**长期前景良好，但非暴利型。**

| 驱动因素 | 方向 | 确定性 |
|----------|------|--------|
| 全球风电装机增长 | 中长期持续增长 | 高确定性（碳中和目标） |
| 存量改造/升级 | 长期稳定需求 | 非常高确定性 |
| 海外市场拓展 | 高增长 | 中等（受地缘政治影响） |
| 非风电领域拓展 | 新增长极 | 低（需验证） |
| 服务收入占比提升 | 利润率改善 | 中等 |

**十年视角推演**：
- 假设全球风电新增装机年均增速5-8%
- 存量改造市场随装机总量线性增长
- 海外收入占比从35%提升至50%+
- 服务收入占比从3.7%提升至10-15%
- 10年后收入规模有望达到40-50亿元（vs 2025年18.78亿元）

## 六、总体结论

### 6.1 段永平"好生意"评分卡

| 评估维度 | 评分 | 简评 |
|---------|:---:|------|
| 差异化 | ★★★★ | 品牌+认证+全球网络构成的复合差异化 |
| 定价权 | ★★★★ | 转换成本高，安全溢价真实存在 |
| 可持续竞争优势 | ★★★★ | 护城河每年在增加，越老越值钱 |
| 简单易懂 | ★★★★★ | 一分钟讲清楚，没有看不懂的角落 |
| 长期前景 | ★★★★ | 碳中和长期受益，海外市场空间大 |
| **综合评级** | **★★★★** | **符合段永平"好生意"标准，但不是顶级"酒类/平台"级生意** |

### 6.2 关键风险与隐忧

1. **单一行业依赖**：96.57%收入来自风电行业，一旦风电政策转向或补贴退坡，影响巨大。虽然碳中和背景下风险可控，但仍需关注。
2. **大客户集中度**：前五大客户占比若过高（需查年报），存在单一客户流失风险。
3. **服务收入占比过低**：3.71%的服务收入意味着"一次性销售"模式尚未转向"持续收费"模式，收入的稳定性和可预测性不如纯服务型生意。
4. **国内竞争加剧**：随着行业景气度提升，可能吸引更多竞争者进入，压缩国内市场份额和毛利率。
5. **海外市场风险**：地缘政治、贸易壁垒、汇率波动可能影响海外业务拓展。

### 6.3 对投资决策的参考意义

从段永平视角看，中际联合**大致符合"好生意"的定义**：

- **优点**：生意简单、护城河真实且在增厚、现金流健康、增长确定性强、管理层专一聚焦
- **不足**：不是顶级商业模式（无网络效应、服务占比低、行业集中）、增长受限于风电行业整体增速

**中际联合更接近"优质制造企业"而非"卓越商业模式"**。这并不意味着不是好投资——如果价格合适，一家好公司+好生意+中等偏上的模式，也可以是一笔好投资。

但段永平可能会说：
> "这是好生意，但不是茅台那种顶级的。关键是价格。"
